# greedy-snake

greedy snake game on terminal.

![greedy-snake](./data/output.gif)

## Installation

```bash
$ npm install greedy-snake -g
```

## Play

```bash
$ greedy-snake [speed]
```

### Options

 * speed -- the speed of of the snake, will be passed to the `setInterval` function
